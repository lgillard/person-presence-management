## Project setup
To use this project you will need vueJS, npm and node. Make sure it is well installed.

After that run next commands
```
$ npm install -g json-server
$ npm install
```

## Project running
Run the API
```
$ npm run api
```
Run the vue application
```
$ npm run serve
```
